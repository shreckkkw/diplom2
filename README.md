# NetGuard — Система детекции аномалий в сетевом трафике

Дипломный проект на Django с нейросетевым автоэнкодером (MLP) для обнаружения аномалий в сетевом трафике.

## Возможности

- **Демо-режим** — генерация тестового трафика с аномалиями (flood, сканирование, утечка)
- **Загрузка CSV** — анализ собственных данных
- **Нейросетевой автоэнкодер** — обучение на нормальном трафике, детекция по ошибке реконструкции
- **История анализов** — сохранение результатов в SQLite
- **Страница «Как это работает»** — описание метода для защиты диплома

## Быстрый старт

```bash
# 1. Установка зависимостей
py -m pip install -r requirements.txt

# 2. Обучение модели (один раз)
py detector/ml/train_model.py

# 3. Миграции базы данных
py manage.py migrate

# 4. Запуск сервера
py manage.py runserver
```

Откройте http://127.0.0.1:8000/

## Принцип работы (кратко)

1. Каждое сетевое соединение описывается 8 признаками (пакеты, байты, порты и т.д.)
2. Автоэнкодер (MLP 8→16→8→16→8) обучается восстанавливать нормальный трафик
3. Для новых записей вычисляется ошибка реконструкции (MSE)
4. Если ошибка выше порога — запись считается аномалией

## Формат CSV

Столбцы: `source_ip`, `dest_ip`, `protocol`, `packet_count`, `byte_volume`, `duration_sec`, `src_port`, `dst_port`, `connections_per_min`, `avg_packet_size`

Пример: `sample_data/traffic_sample.csv`
